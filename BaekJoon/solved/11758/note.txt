# 01
처음 > 기울기??

시계 반시계는 기울기보다는
P1과 P2가 그린 직선의 아래에 있냐 위에 있냐의 개념

따라서 해당 직선 을 f(x, y) = 0 꼴로 나타낸 다음에
f(x3, y3) 를 0과 비교해서 큰지 작은지

라그랑쥬 방정식 쓰면 함수 바로 나올듯?

직선 그 자체 뿐만 아니라 P2도 고려해줘야함
뭐가 P2에 따라 결과가 바뀜
이 부분은 함수를 ax+by+c = 0 꼴 에서 f(x, y)를 어떻게 설정할지 고민하면 알아서 결정됨

a의 계수 : y2-y1
b의 계수 : x2-x1

둘 중 하나에 음수를 줘야함. -> 어디에 음수를 주냐에 함수 위에가 -가 될지, 함수 아래가 -가 될지 다름

b의 계수에 양수를 줘야, 같은 x에 대해 기준y보다 큰 y를 주었을 때 변화량만큼 차이가 남.
즉 변화량이 양수였으면 양수만큼 차이, 음수였으면 음수만큼 차이

수식 = (y 변화량) * x - (x 변화량) * y  + c = 0;

Case  01) x변 + / y변 + :  같은 x 변화량에 대해 y가 크면 반시계 -> 통과
Case  01) x변 + / y변 - :  같은 x 변화량에 대해 y가 크면 반시계 -> 통과
Case  01) x변 - / y변 + :  같은 x 변화량에 대해 y가 크면 시계 -> 통과
Case  01) x변 - / y변 - :  같은 x 변화량에 대해 y가 크면 시계 -> 통과
