## 알고리즘 
### 전략
- Brute Force
- Depth First Search (DFS) / Breadth First Search (BFS)
- Greedy
- Dynamic Programming (DP)

    + Longest Increasing Subsequence (LIS)
- **Knapsack**

### 그래프 관련
- DijkStra
- Floyd-Warshall
- Topological Sort
- Union-Find
- **Strongly Connected Component (SCC)**

### 문자열 관련
- **KMP**
- **Aho-Corasick**

### 수학 관련
